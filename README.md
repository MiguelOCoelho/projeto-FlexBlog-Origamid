# FlexBlog

Blog moderno **FlexBlog**, desenvolvido como projeto prático do curso **CSS Flexbox da Origamid**. Foco em criar **layout 100% responsivo** usando apenas propriedades Flexbox (sem valores fixos ou float), aplicando **flex-grow**, **flex-basis**, **Flex Container** e mais.

## **Funcionalidades Principais**
- **Página Inicial**: Hero com "Novos valores e propriedades do CSS", call-to-action e destaques.
- **Seções**: Sobre, Produtos, Preço, Qualidade – todas com Flexbox layouts.
- **Design Responsivo**: **100% responsivo** celular/desktop via Flexbox puro.
- **UI Flexbox**: Header com menu horizontal, footer, cards produtos alinhados perfeitamente.
- **Estrutura Limpa**: Semântica HTML5 + CSS modular.

## **Tecnologias e Skills Aplicadas**

| **Skill** | **Descrição** | **Exemplo no Projeto** |
|-----------|---------------|------------------------|
| **HTML5** | Estrutura semântica completa | `index.html`, seções semânticas |
| **CSS3 Flexbox** | **Flex Container**, **flex-grow**, **flex-basis** | Header, grid produtos, alinhamentos |
| **Layout Responsivo** | Flexbox adaptável sem media queries pesadas | Mobile: colunas empilhadas; Desktop: side-by-side |
| **CSS Modular** | Arquivos organizados por seção | `estilo.css` com todas propriedades Flexbox |
| **Git/GitHub** | Versionamento + Pages ativo | "Primeiro commit", "Site FlexBlog concluído" |
